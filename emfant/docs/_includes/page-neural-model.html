Spinal Cord Circuit Overview

Spinal cord circuit model with tactile feedback (from Yamada et al., 2016)

The neural model implements a biologically plausible spinal cord circuit that generates spontaneous fetal movements without explicit motor commands. Movement emerges from the dynamics of body-environment interaction.

Circuit Components

α Motor Neuron

Output commands to muscles. Receives input from BVP oscillator, spindle Ia, and tendon Ib.

γ Motor Neuron

Controls muscle spindle sensitivity. Receives input from α motor neuron.

S0 Neuron

Processes afferent sensory signals from spindle Ia with low-pass filtering.

Neural Oscillator (BVP)

Bonhoeffer-Van der Pol oscillator generates rhythmic motor patterns. Receives input from S0 neurons and brain control signal. Parameters based on Kuniyoshi & Sangawa (2006).

Brain Control

Simple control signal (I_control) modulates BVP oscillator activity. Can be used to study effects of descending control on motor patterns.

Proprioceptors

Muscle Spindle (Ia Afferent)

Senses muscle length and velocity. Model based on He et al., with dynamic and static components responding to gamma motor neuron activation.

  • Dynamic sensitivity to length changes
  • Static response to absolute length
  • Gamma-dynamic activation control

Golgi Tendon Organ (Ib Afferent)

Senses muscle tension/force. Provides inhibitory feedback to prevent excessive muscle activation.

  • Responds to muscle force
  • Peak force normalized output
  • Inhibitory connection to α neuron

Signal Delays

The circuit implements biologically realistic signal transmission delays:

Pathway Delay Description
Tactile → Spinal 10 ms Cutaneous afferent transmission
Spindle → Spinal 10 ms Ia afferent transmission
Tendon → Spinal 10 ms Ib afferent transmission
Spindle S0 → BVP 50 ms Processed signal to oscillator
BVP → α Neuron 50 ms Oscillator output delay
α → Muscle 10 ms Efferent motor command
γ → Spindle 10 ms Gamma motor neuron effect

Gain Parameters

Key gain parameters controlling circuit behavior:

Connection Gain Notes
Spindle → S0 0.25 Ia to S0 neuron
Spindle → α 0.04 Stretch reflex (Sangawa et al.)
Tendon → α -1.0 Inhibitory Ib feedback
S0 → BVP 1.0 Sensory input to oscillator
BVP → α 1.0 Oscillator to motor output
α → Spindle 0.5 Alpha-gamma coactivation
γ → Spindle 1.0 Gamma sensitivity control
α → γ 0.5 Alpha to gamma coupling

Muscle Activation Clipping

Activation Range

Based on inverse kinematics from motion capture data, the maximum muscle activation is clipped to 0.15 (15%). Actual infant movements show activation levels of 0.10-0.20 maximum, with average around 0.05. This prevents unrealistic high-force movements in simulation.
