Skeletal System

The skeleton model is created by scaling OpenSim adult models based on measured infant data:

Upper Body

Arm model from Saul et al. and McFarland et al., including shoulder complex with scapula coupling.

Lower Body

Leg model from Lai et al., featuring patella coupling joint for realistic knee mechanics.

Coupling Joint System

Unlike standard MuJoCo, EMFANT implements coupling joints where auxiliary joints move cooperatively with main joints. This biological coordination is constrained by ligaments, muscles, and skin.

Implementation

The coupling joint uses MuJoCo's equality constraint feature:

  1. Calculate target value of auxiliary joint based on main joint angle
  2. Update the polycoef in equality element
  3. MuJoCo's constraint solver generates acceleration toward target state
  4. Repeat for each simulation step

Muscle System

186 muscles with attachment positions from OpenSim models. The muscle activation model is based on He et al. (2001).

Muscle Force Model

Incorporates force-length relationship, force-velocity relationship, and tendon elasticity. Maximum isometric force is scaled based on infant body mass ratio.

Muscle Path Constraints (Wrap Objects)

Wrap objects constrain muscle paths to prevent incorrect force transmission:

(a) Default state

(b) Without constraint: wrong force direction

(c) With constraint: correct rotational force

Tactile System

Sensor Distribution

1,000 tactile sensors distributed on the skin based on two-point discrimination density data from Weinstein (1968).

Merkel Cell Model

Low-pass filter with 50 Hz cutoff frequency simulates Merkel cell (SA-I mechanoreceptor) responses.

Soft Uterine Model

The uterine shape is derived from MRI data (FEMONUM FETUS project, 34.5 weeks of amenorrhea).

Soft Body Implementation

Since MuJoCo is a rigid body simulator, soft objects are implemented using equality constraints:

  1. Divide target mesh into pieces (300 vertices, 827 edges)
  2. Create 3 hinge joints and 3 slide joints at each vertex
  3. Generate tendons for each edge between vertices
  4. Apply equality constraints to maintain original positions
